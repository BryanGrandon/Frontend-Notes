### `.insertAdjacent`

- .insertAdjacentElement( position, element )
- .insertAdjacentHTML( position, html )
- .insertAdjacentText( position, text )

| Position      | Where it is added |
| ------------- | ----------------- |
| `beforebegin` | Previous brother. |
| `afterend`    | Next brother.     |
| `afterbegin`  | First son.        |
| `beforeend`   | Last son.         |
